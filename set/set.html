集团简介
当前位置：首页 > 走进煤销 > 集团简介

晋能集团有限公司（以下简称“ 晋能集团”）2013年5月，是由原山西煤炭运销集团与山西国际电力集团合并重组，集煤炭、电力、清洁能源、贸易物流、装备制造、房地产等产业于一体的大型综合能源集团。作为山西省省属国有重点骨干企业，2015年《财富》世界500强列第382位。

集团重组成立以来，围绕建设现代综合能源集团目标，积极推进产业结构调整和转型升级，到2015年底，集团煤炭生产矿井50座，联合试运转矿井12座，产能7100万吨／年；选煤厂23座，洗选能力4785万吨／年；燃煤发电装机458.2万千瓦，在建装机477万千瓦；光伏、风电新能源发电装机167.04万千瓦，在建装机75.6万千瓦，光伏电池生产能力50万千瓦，光伏组件生产能力60万千瓦；煤炭贸易量1.33亿吨；多元产业形成装备制造、房地产开发、旅游酒店、有色金属、高效绿色农业等板块。集团资产总额达到2300亿元。

“十三五”我国经济发展进入了新常态，表现出速度变化、结构优化、动力转换的特点，集团顺应改革大局，提出了“1255”发展战略：即“以煤、电、清洁能源为基础，以贸易物流为支撑，整合优化多元产业，围绕一流综合能源集团目标，实现发展理念、发展方式转变，推进安全发展、创新发展、绿色发展、协调发展、共享发展，建设平安型晋能、创新型晋能、绿色型晋能、效益型晋能、和谐型晋能。”概括起来就是“围绕一个目标，实现两个转变，推进五大发展，建设五型晋能”。

煤炭产业按照开采、加工、运输、使用全过程绿色化、清洁化，电力产业按照超低排放、节能高效和清洁化综合利用，清洁能源产业按照高端化、规模化，贸易物流产业按照现代化、市场化、信息化和物流供应链管理，多元产业按照整合优化，加快转型，打造亮点的产业发展思路，力争到“十三五”末，实现煤炭产能1亿吨，火电装机1500万千瓦，光伏、风电装机500万千瓦，煤炭贸易量2亿吨，多元产业形成装备制造、房地产、酒店旅游等板块，实现年营业收入1500亿元，利税100亿元。
